To filter meeting invitations into a specific folder in Outlook Web, follow these simple steps.

Option 1: The Simplest Steps (Manual Setup)
Open Settings: Go to outlook.office.com and click the Gear icon (Settings) in the top right corner.
Navigate to Rules: Go to Mail > Rules and click + Add new rule.
Name your rule: Type something like Meeting Invites.
Add a Condition:
Open the dropdown and select Type.
In the second dropdown, select Invitation.
Add an Action:
Select Move to.
Click Select a folder and choose New folder.
Type Calendar Invites (or your preferred name) and press Enter.
Save: Click Save at the bottom.
